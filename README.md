⚙️ Installation

1️⃣ Clone the Repository

git clone https://github.com/yourusername/sukuna-md
cd sukuna-md

2️⃣ Install Dependencies

npm install

3️⃣ Start the Bot

node index.js

---

🔐 Pair Code Login

Run the bot and request your pairing code.

Then open WhatsApp:

Linked Devices → Link with Phone Number

Enter the generated code to connect the bot.

No QR scan required.
